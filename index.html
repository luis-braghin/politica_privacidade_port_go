Política de Privacidade

Última atualização: 1 de abril de 2026

O Portaria Go é um sistema de gestão de portaria para condomínios residenciais. Esta política descreve como coletamos, usamos e protegemos os dados dos usuários.

1. Dados Coletados

O app coleta os seguintes dados para fins de funcionamento:

  • Nome – para identificar o morador no sistema de notificações.
  • Número de telefone – para envio de notificações via WhatsApp sobre entregas e comunicados do condomínio.
  • Apartamento e torre – para vincular o morador à unidade residencial.
  • Token de notificação push (FCM) – gerado automaticamente pelo dispositivo para receber notificações push de entregas e interfone digital.

Esses dados são fornecidos voluntariamente pelo próprio usuário durante o cadastro ou coletados automaticamente pelo dispositivo (token push).

2. Uso dos Dados

Os dados coletados são utilizados exclusivamente para:

  • Enviar notificações de entregas e encomendas via WhatsApp.
  • Enviar comunicados oficiais do condomínio.
  • Enviar notificações push de interfone digital (Firebase Cloud Messaging).
  • Identificar o morador no painel interno da portaria.

Não utilizamos os dados para fins de publicidade, rastreamento ou qualquer outra finalidade além das descritas acima. O app não exibe anúncios e não rastreia usuários entre aplicativos.

3. Assistente Virtual (IA)

O app inclui um assistente virtual que responde perguntas dos moradores sobre as regras e regulamentos do condomínio.

Como funciona: quando o usuário digita uma pergunta, apenas o texto da pergunta é processado. Os documentos do condomínio são indexados via embeddings (OpenAI) para busca semântica, e a resposta é gerada pela Anthropic (Claude AI). Nenhum dado pessoal (nome, telefone, apartamento ou qualquer informação de identificação) é transmitido junto com a pergunta.

Terceiros envolvidos:

  • OpenAI – embeddings para busca semântica: https://openai.com/privacy
  • Anthropic, PBC – geração de respostas: https://www.anthropic.com/privacy

As perguntas não são armazenadas pelo nosso sistema após o processamento.

4. Notificações Push

O app utiliza Firebase Cloud Messaging (FCM) para enviar notificações push de entregas e interfone digital. Um token de identificação do dispositivo é armazenado no nosso servidor para possibilitar o envio. Este token não contém dados pessoais e pode ser revogado a qualquer momento desinstalando o app.

Terceiro envolvido: Google Firebase – https://firebase.google.com/support/privacy

5. Compartilhamento de Dados

Os dados pessoais (nome e telefone) não são compartilhados com terceiros, exceto:

  • Evolution API / WhatsApp – para envio das mensagens de notificação ao número cadastrado pelo próprio usuário.
  • Firebase Cloud Messaging (Google) – para envio de notificações push (apenas o token do dispositivo, sem dados pessoais).

Não vendemos, alugamos ou comercializamos dados de usuários.

6. Armazenamento

Os dados são armazenados em banco de dados PostgreSQL gerenciado pelo Supabase (infraestrutura em nuvem com criptografia em repouso e em trânsito). O acesso é restrito à equipe da portaria e à administração do condomínio, protegido por autenticação e políticas de segurança em nível de linha (Row Level Security).

7. Exclusão de Conta e Dados

O usuário pode solicitar a exclusão completa de sua conta e dados pessoais a qualquer momento:

  • Enviando um e-mail para contato@portariago.app com nome, telefone e condomínio.
  • Ou solicitando diretamente ao administrador do condomínio pelo app.

Ao solicitar a exclusão:

  • A conta será desativada em até 7 dias úteis.
  • Todos os dados pessoais (nome, telefone, token push, credenciais, preferências) serão removidos.
  • Registros históricos de notificações podem ser mantidos de forma anonimizada por até 12 meses para fins de auditoria do condomínio.

Para mais detalhes, acesse: Página de exclusão de conta.

8. Direitos do Usuário

O usuário pode, a qualquer momento:

  • Solicitar a exclusão completa de seus dados (veja seção 7).
  • Atualizar seu número de telefone ou preferências de notificação.
  • Revogar permissões de notificação push nas configurações do dispositivo.

9. Segurança

Adotamos medidas técnicas e organizacionais para proteger os dados contra acesso não autorizado, perda ou alteração indevida, incluindo:

  • Criptografia em trânsito (HTTPS/TLS) e em repouso.
  • Row Level Security (RLS) no banco de dados, garantindo isolamento entre condomínios.
  • Autenticação obrigatória para acesso ao sistema.
