Política de Privacidade

Última atualização: 25 de fevereiro de 2026

O Portaria Go é um sistema de gestão de portaria para condomínios residenciais. Esta política descreve como coletamos, usamos e protegemos os dados dos usuários.

1. Dados Coletados

O app coleta os seguintes dados para fins de funcionamento:

  • Nome – para identificar o morador no sistema de notificações.
  • Número de telefone – para envio de notificações via WhatsApp sobre entregas e comunicados do condomínio.

Esses dados são fornecidos voluntariamente pelo próprio usuário durante o cadastro.

2. Uso dos Dados

Os dados coletados são utilizados exclusivamente para:

  • Enviar notificações de entregas e encomendas via WhatsApp.
  • Enviar comunicados oficiais do condomínio.
  • Identificar o morador no painel interno da portaria.

Não utilizamos os dados para fins de publicidade, rastreamento ou qualquer outra finalidade além das descritas acima.

3. Assistente Virtual (IA)

O app inclui um assistente virtual chamado "ChatGPT do Imperatriz", que responde perguntas dos moradores sobre as regras e regulamentos do condomínio.

Como funciona: quando o usuário digita uma pergunta, apenas o texto da pergunta é enviado à API da Anthropic (Claude) para processamento. Nenhum dado pessoal (nome, telefone, apartamento ou qualquer informação de identificação) é transmitido junto com a pergunta.

Terceiro envolvido: Anthropic, PBC – https://www.anthropic.com/privacy

As perguntas não são armazenadas pelo nosso sistema após o processamento.

4. Compartilhamento de Dados

Os dados pessoais (nome e telefone) não são compartilhados com terceiros, exceto:

  • Evolution API / WhatsApp – para envio das mensagens de notificação ao número cadastrado pelo próprio usuário.

Não vendemos, alugamos ou comercializamos dados de usuários.

5. Armazenamento

Os dados são armazenados em planilhas Google Sheets sob controle exclusivo do administrador do condomínio. O acesso é restrito à equipe da portaria e à administração.

6. Direitos do Usuário

O usuário pode, a qualquer momento:

  • Solicitar a exclusão de seus dados entrando em contato com a administração do condomínio.
  • Atualizar seu número de telefone ou preferências de notificação.

7. Segurança

Adotamos medidas técnicas e organizacionais para proteger os dados contra acesso não autorizado, perda ou alteração indevida.
