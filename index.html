Portaria Go

Política de Privacidade

Última atualização: 22 de fevereiro de 2026

1. Sobre este aplicativo

O Portaria Go é um aplicativo de gestão de portaria desenvolvido para condomínios residenciais. Ele permite que porteiros registrem e notifiquem moradores sobre entregas, além de enviar comunicados oficiais do condomínio via WhatsApp.

2. Dados coletados

O aplicativo coleta exclusivamente os seguintes dados dos moradores cadastrados:

  • Nome completo — utilizado para identificação nas notificações
  • Número de telefone — utilizado para envio de notificações via WhatsApp

Esses dados são fornecidos voluntariamente pelo próprio morador ou pelo síndico/administrador do condomínio no momento do cadastro.

3. Finalidade do uso dos dados

Os dados coletados são utilizados exclusivamente para:

  • Envio de notificações de entregas e encomendas via WhatsApp
  • Envio de comunicados oficiais do condomínio
  • Identificação do morador na interface de gestão da portaria

4. Armazenamento dos dados

Os dados são armazenados em planilhas Google Sheets vinculadas à conta do condomínio contratante. O Portaria Go não mantém servidores próprios de armazenamento de dados pessoais.

5. Compartilhamento de dados

Os dados não são compartilhados com terceiros para fins comerciais, publicitários ou quaisquer outros fins além dos descritos nesta política. O acesso é restrito aos administradores do condomínio contratante.

6. Retenção dos dados

Os dados permanecem armazenados enquanto o morador estiver cadastrado no sistema. O morador pode solicitar a exclusão dos seus dados a qualquer momento entrando em contato com a administração do seu condomínio.

7. Direitos do usuário (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018), o usuário tem direito a:

  • Acessar seus dados cadastrados
  • Corrigir dados incompletos ou desatualizados
  • Solicitar a exclusão de seus dados
  • Revogar o consentimento para recebimento de notificações

Para exercer esses direitos, entre em contato com a administração do seu condomínio.

8. Segurança

Os dados são transmitidos via HTTPS e armazenados nos servidores do Google, que seguem padrões internacionais de segurança da informação.
